Name
-EZFlow Example

Description
-This is an example of an IVR system that uses a voice menu to guide callers through a selection of company, product and support information. The key processes in this system are answering the phone, playing a greeting, prompting the caller to select an option, collecting digits, processing the selection (by playing the appropriate message) and then terminating the call.

Steps
-Answer the call.
-Play an initial greeting.
-Present a menu of options.
-Depending on the option chosen, branch and play one of two greetings, or hang up.
-If one of the greetings are played, return to the menu

Implementation
-The LineGroup control "PhoneAns" detects the incoming call, and takes the line OffHook.
-The PlayGreeting "IntroGreeting" plays an initial greeting.
-The GetDigits "GetCustInfo" presents the options ("1 for product info, 2 for support, # to leave"). Based on the input, the call then branches.
-PlayGreeting "ProductInfo" is reached if the caller selected "1" in the GetDigits.
-PlayGreeting "SupportInfo" is reached if the caller selected "2" in the GetDigits.
-When either PlayGreeting is finished, the call returns to "GetCustInfo".
-"OnHook1" is reached if the caller selected "#".

Notes
A full, detailed tutorial on how to build this application can be found in the ‘Getting Started’ document in the 'docs' directory.

This example was created in C# (Visual Studio 2003)
